12. Lookups

LookupType

Defines configurable dropdown types. Fields: code (LookupTypeCode enum — unique), name, description, isSystem.

LookupValue

Individual dropdown options. Fields: lookupTypeId (FK → LookupType), shopId, code, name, color, sortOrder, isActive, isDefault. Soft-deletable. Used by Product categories, Expense categories, Income categories.
