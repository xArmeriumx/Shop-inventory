1. เริ่มต้นใช้งาน

1.1 สมัครสมาชิก

  1. เข้าหน้า สมัครสมาชิก — กรอกชื่อ อีเมล รหัสผ่าน
  2. กด "สมัคร"
  3. ระบบจะนำไปหน้า สร้างร้านค้า

1.2 สร้างร้านค้า

  1. กรอกชื่อร้าน ที่อยู่ เบอร์โทร เลขผู้เสียภาษี (ถ้ามี)
  2. กด "สร้างร้าน"
  3. เข้าสู่แดชบอร์ดหลักได้ทันที

1.3 เข้าสู่ระบบ

กรอกอีเมลและรหัสผ่าน → กด "เข้าสู่ระบบ"

2. แดชบอร์ด

หน้าแรกหลังเข้าสู่ระบบ แสดงข้อมูลสรุป:

  • การ์ด KPI: ยอดขายวันนี้, รายได้, ค่าใช้จ่าย, กำไร
  • กราฟยอดขาย: แสดง 7 วันล่าสุด
  • รายการขายล่าสุด: 5 รายการ
  • สินค้าสื่อกต่ำ: แจ้งเตือนเมื่อสต็อกต่ำกว่า minStock

3. สินค้า

3.1 เพิ่มสินค้า

  1. ไปที่ สินค้า → เพิ่มสินค้า
  2. กรอก: ชื่อ, SKU (ถ้ามี), หมวดหมู่, ราคาต้นทุน, ราคาขาย, จำนวนสต็อก, สต็อกต่ำสุด
  3. อัปโหลดรูปสินค้า (ถ้ามี)
  4. เลือกซัพพลายเออร์ (ถ้ามี)
  5. กด "บันทึก"

3.2 แก้ไขสินค้า

คลิกที่สินค้า → แก้ไขข้อมูล → กด "บันทึก"

หมายเหตุ: ระบบใช้ Optimistic Locking ถ้ามีคนอื่นแก้ไขพร้อมกัน จะแจ้งเตือน

3.3 ปรับสต็อก

  1. คลิกที่สินค้า → "ปรับสต็อก"
  2. เลือกประเภท: เพิ่ม / ลด / ตั้งค่า
  3. ใส่จำนวนและเหตุผล (จำเป็น)
  4. กด "บันทึก"

3.4 นำเข้า CSV

อัปโหลดไฟล์ CSV ตามรูปแบบที่กำหนด ระบบจะสร้างสินค้าหลายรายการพร้อมกัน

3.5 ลบสินค้า

สินค้าจะ ซ่อน (soft delete) ไม่ถูกลบจริง ถ้ามีรายการขายที่ยังไม่ยกเลิก จะไม่สามารถลบได้

4. ขายหน้าร้าน (POS)

4.1 สร้างรายการขาย

  1. ไปที่ POS
  2. ค้นหาและเลือกสินค้า → เพิ่มลงตะกร้า
  3. ปรับจำนวน / ใส่ส่วนลดรายชิ้น (ถ้ามี)
  4. ใส่ส่วนลดรวมบิล (ถ้ามี) — เลือก % หรือ จำนวนเงิน
  5. เลือกช่องทางการขาย (Walk-in, Shopee, Lazada ฯลฯ)
  6. เลือกลูกค้า (ถ้ามี)
  7. เลือกวิธีชำระเงิน: เงินสด / โอนเงิน / เครดิต
  8. ถ้าเลือก "โอนเงิน" → แนบรูปสลิป
  9. กด "ยืนยันการขาย"

4.2 ผลลัพธ์

  • สร้างเลขที่ใบเสร็จ (INV-00001) อัตโนมัติ
  • ตัดสต็อกทันที
  • ถ้าเงินสด → สถานะ "ยืนยันแล้ว" ทันที
  • ถ้าโอนเงิน → สถานะ "รอตรวจสอบ"
  • สามารถพิมพ์ใบเสร็จขนาด 80mm ได้

5. จัดการรายการขาย

5.1 ดูรายการขาย

ไปที่ รายการขาย — ค้นหาตามเลขใบเสร็จ, ชื่อลูกค้า, กรองวันที่/ช่องทาง/สถานะ

5.2 ยืนยันสลิป

สำหรับผู้มีสิทธิ์ PAYMENT_VERIFY: ดูรูปสลิป → กด "อนุมัติ" หรือ "ปฏิเสธ" พร้อมเหตุผล

5.3 ยกเลิกการขาย

  1. เปิดรายการขาย → กด "ยกเลิก"
  2. ระบุเหตุผล (บันทึกผิด / ลูกค้าร้องขอ / สินค้าชำรุด / อื่นๆ)
  3. ระบบจะ คืนสต็อกอัตโนมัติ และยกเลิกการจัดส่ง (ถ้ามี)

6. การซื้อสินค้า

6.1 บันทึกการซื้อ

  1. ไปที่ การซื้อ → เพิ่มการซื้อ
  2. เลือกซัพพลายเออร์ (ถ้ามี)
  3. เลือกสินค้า ใส่จำนวน ราคาต้นทุน
  4. เลือกวิธีชำระเงิน
  5. กด "บันทึก"

ผลลัพธ์: เพิ่มสต็อกอัตโนมัติ + อัปเดตราคาต้นทุนสินค้า

6.2 ยกเลิกการซื้อ

เลือกเหตุผลและยืนยัน ระบบจะตรวจสอบว่ามีสต็อกเพียงพอก่อนหักคืน

7. การจัดส่ง

7.1 สร้างรายการจัดส่ง

  1. ไปที่ การจัดส่ง → เพิ่ม
  2. เลือกบิลขายที่ยังไม่มีการจัดส่ง
  3. กรอกชื่อผู้รับ ที่อยู่ เบอร์โทร
  4. เลือกที่อยู่ที่บันทึกไว้ (ถ้ามี)
  5. กด "สร้าง" → สถานะเริ่มต้นเป็น "รอจัดส่ง"

7.2 อัปเดตสถานะ

รอจัดส่ง → ส่งแล้ว → ส่งถึงแล้ว / ส่งคืน (สถานะเป็น State Machine ไม่สามารถข้ามขั้นได้)

7.3 OCR จับคู่พัสดุ

สแกนใบปะหน้าพัสดุ → ระบบจับคู่กับออเดอร์จากเบอร์โทรลูกค้าอัตโนมัติ

8. คืนสินค้า

8.1 สร้างรายการคืน

  1. ไปที่ คืนสินค้า → สร้าง
  2. เลือกบิลขายต้นทาง
  3. เลือกสินค้าที่จะคืนและจำนวน (ระบบแสดงจำนวนที่คืนได้สูงสุด)
  4. ระบุเหตุผลและวิธีคืนเงิน
  5. กด "บันทึก"

ผลลัพธ์: คืนสต็อก + ปรับยอดบิลต้นทาง + สร้างเลขที่คืน (RET-00001)

9. ลูกค้าและซัพพลายเออร์

ลูกค้า

  • เพิ่ม/แก้ไข: ชื่อ เบอร์โทร อีเมล ที่อยู่ เลขภาษี หมายเหตุ
  • จัดการที่อยู่จัดส่ง: เพิ่มได้หลายที่อยู่ ตั้งค่าเริ่มต้น
  • โปรไฟล์: ดูประวัติซื้อ ยอดซื้อรวม พัสดุ

ซัพพลายเออร์

  • เพิ่ม/แก้ไข: ชื่อ รหัสซัพพลายเออร์ ผู้ติดต่อ เบอร์โทร อีเมล ที่อยู่ เลขภาษี
  • โปรไฟล์: ดูประวัติการซื้อ ยอดรวม

10. รายรับ-รายจ่าย

บันทึกรายรับ/รายจ่ายที่ไม่เกี่ยวกับการขาย เช่น ค่าเช่า, ค่าน้ำ-ไฟ, รายได้จากบริการ

  • เลือกหมวดหมู่ ใส่จำนวนเงิน คำอธิบาย วันที่
  • แนบรูปใบเสร็จ (ถ้ามี)
  • สรุปรายเดือนอัตโนมัติ

11. รายงาน

รายงานรายละเอียด
รายงานหลักยอดขาย ต้นทุน กำไร สรุปรายวัน ช่วงวันที่กำหนดเอง
งบกำไรขาดทุน (P&L)รายได้ - COGS - ค่าใช้จ่าย + รายได้อื่น = กำไรสุทธิ
สินค้าขายดีจัดอันดับตามรายได้
กำไรตามสินค้ารายได้ ต้นทุน กำไร margin ต่อสินค้า
เปรียบเทียบช่วงเวลาเดือนนี้ vs เดือนที่แล้ว (% เปลี่ยนแปลง)
มูลค่าสต็อกมูลค่าสินค้าคงคลัง ณ ราคาต้นทุน + สินค้าค้างนาน (90+ วัน)
Inventory Turnoverอัตราหมุนเวียนสินค้า + จำนวนวันขาย
ยอดขายตามช่องทางWalk-in, Shopee, Lazada ฯลฯ
จัดอันดับลูกค้าลูกค้าที่ซื้อมากที่สุด
ค่าใช้จ่ายตามหมวดหมู่สัดส่วนค่าใช้จ่ายแต่ละหมวด

ส่งออก CSV ได้ทุกรายงาน

12. ตั้งค่า

12.1 โปรไฟล์ร้าน

ชื่อร้าน ที่อยู่ เบอร์โทร เลขภาษี คำนำหน้าใบเสร็จ

12.2 จัดการทีม

เชิญสมาชิก → กำหนดบทบาท → แก้ไข/ลบ (ไม่สามารถลบเจ้าของได้)

12.3 จัดการบทบาท

สร้างบทบาทใหม่ → เลือกสิทธิ์จาก 52 รายการ → บันทึก

12.4 Lookup Values

ปรับแต่งหมวดหมู่สินค้า หมวดหมู่ค่าใช้จ่าย หมวดหมู่รายรับ ฯลฯ

13. สแกนใบเสร็จ AI

  1. ถ่ายรูปใบเสร็จ
  2. อัปโหลด → ระบบใช้ Tesseract.js อ่านข้อความ
  3. Groq AI วิเคราะห์ข้อความ → แยกรายการสินค้า จำนวน ราคา
  4. ตรวจสอบข้อมูลที่ AI แยกออกมา → กด "บันทึก"

หมายเหตุ: ฟีเจอร์นี้ต้องตั้งค่า GROQ_API_KEY
